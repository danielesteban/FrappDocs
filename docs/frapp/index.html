1. Frapp Public A.P.I.

Events

window.addEventListener('frapp.init')

Triggered to the current Frapp window when the modules are loaded & the A.P.I. is ready.

event.detail can contain the optional launch params for the current Frapp (params) and the current session user (session)

window.addEventListener('frapp.signin', callback)

Trigered to all Frapp windows when the user signs in.

window.addEventListener('frapp.signout', callback)

Trigered to all Frapp windows when the user signs out.

Methods

FRAPP.close()

Closes the current Frapp window

FRAPP.create(params, callback)

  • @params: { name : 'AppName' }

Creates a new Frapp

FRAPP.edit(frapp)

  • @frapp: Frapp (repo)

Launches frapp code on the FrappEditor

FRAPP.installed(callback)

Returns an array of installed Frapps

FRAPP.menu()

Loads the FrappsMenu

FRAPP.load(frapp, params, closeCaller)

  • @frapp: { repository : { type: 'git', url : 'https://github.com/{author}/{frapp}.git' } }
  • @params: Optional launch params (See: frap.init event)
  • @closeCaller: (bool) It closes the current Frapp when loading the new one when true.

Loads a Frapp. It also downloads and installs it first if is not alredy on the Frapps folder.

FRAPP.readFile(filePath, callback)

Reads a File (relative paths inside Frapps folder only)

FRAPP.reload()

Reloads the current Frapp

FRAPP.saveFile(filePath, data, callback)

Saves a File (relative paths inside Frapps folder only)

FRAPP.setTitle(title)

Sets the Frapp window title

FRAPP.signin(auth, callback)

  • @auth: { username : '{username}', password : '{password}'}

Signs the user into GitHub

FRAPP.signout()

Signs out the user from GitHub

FRAPP.showDevTools()

Shows the DevTools for the current Frapp

FRAPP.storage()

Returns a key value storage on the current Frapp's namespace

FRAPP.version()

Returns the Frapp engine version & the current Frapp version
